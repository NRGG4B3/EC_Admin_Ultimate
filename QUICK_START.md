# Quick Setup Flowchart

## Choose Your Setup Method

```
START
  │
  ├─► First time setup?
  │   └─► Run: setup-quick.bat ✓ (Recommended)
  │
  ├─► Want detailed output?
  │   └─► Run: setup.bat
  │
  └─► Custom setup?
      └─► Manual npm commands
```

## What Setup Does

```
setup-quick.bat
│
├─ ✓ Check Node.js is installed
├─ ✓ Check npm is installed
│
├─ ✓ Install UI dependencies (npm install)
├─ ✓ Build UI for production (npm run build)
├─ ✓ Create ui/dist/ folder
│
├─ ✓ Install API dependencies
│
├─ ✓ Copy files to release folder
├─ ✓ Copy ui/dist to release
│
├─ ✓ Create release.zip
│
├─ ✓ Create .env configuration file
│
└─ ✓ Verify everything is ready
    └─ Success! UI built and packaged
```

## After Setup: Starting the Server

```
.env File (EDIT FIRST!)
├─ DB_HOST=localhost
├─ DB_USER=root
├─ DB_PASSWORD=your_password
└─ PORT=30121

   ↓

host/start.bat
├─ Starts Node.js API server
├─ Listens on port 30121
├─ Connects to database
└─ Serves UI files

   ↓

http://localhost:30121
├─ Loads UI from ui/dist/
├─ Connects to API
└─ Admin panel opens ✓
```

## Troubleshooting Quick Reference

| Problem | Solution |
|---------|----------|
| "npm not found" | Install Node.js from nodejs.org |
| "UI build failed" | Rerun setup, check internet connection |
| "UI is blank" | Verify ui/dist/index.html exists, rebuild |
| "Port in use" | Stop other servers or change PORT in .env |
| "DB connection failed" | Update credentials in .env |

## Commands You Need

```bash
# Initial setup (one time)
cd host
setup-quick.bat

# After setup: Edit configuration
Edit: host/node-server/.env

# Start server (whenever you want to run)
host/start.bat

# Stop server
host/stop.bat

# Deploy to FiveM server
Upload: release.zip
Extract in: resources/
Add to server.cfg: ensure EC_Admin_Ultimate
```

## What Gets Built

```
npm install
└─ Downloads React, Vite, TypeScript, etc.
   (~500MB in node_modules/)

npm run build
└─ Compiles React source → JavaScript/CSS
   └─ Creates optimized ui/dist/
       ├─ index.html (~1KB)
       ├─ assets/
       │   ├─ index-*.js (minified React bundle)
       │   └─ index-*.css (compiled Tailwind)
       └─ Total: ~500KB-2MB (minified & gzipped)
```
